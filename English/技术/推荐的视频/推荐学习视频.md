# 推荐学习视频

## YouTube技术频道推荐

### 1. Kodeco (原Ray Wenderlich)

**频道简介：** 业界最知名的iOS开发教程频道之一，提供高质量的Swift、UIKit、SwiftUI教程。

**订阅地址：** https://www.youtube.com/@KodecoHQ

**推荐学习视频：**

| 视频标题 | 时长 | 学习内容 |
|----------|------|----------|
| SwiftUI Tutorial for Beginners | 45min | SwiftUI基础，适合入门 |
| UIKit Fundamentals | 60min | UIKit核心概念 |
| Auto Layout Tutorial | 40min | 自动布局详解 |
| Networking in Swift | 35min | iOS网络编程 |
| Core Data Tutorial | 50min | 数据持久化 |
| Swift Concurrency | 45min | async/await并发编程 |
| Unit Testing in Swift | 30min | 单元测试入门 |
| Git and GitHub for Developers | 40min | 版本控制基础 |

**学习建议：** 从SwiftUI Tutorial开始，循序渐进。

## 学习计划建议

### 入门阶段（1-2月）

**每日任务：**
1. 看1个CodeWithChris短视频（10-15分钟）
2. 跟读模仿发音
3. 记录5个新单词

**推荐顺序：**
1. Swift Programming Tutorial
2. How to Make an App
3. Xcode Tips and Tricks
4. Auto Layout Tutorial

---

### 进阶阶段（3-4月）

**每日任务：**
1. 看1个Kodeco或Paul Hudson教程（30分钟）
2. 做笔记整理表达
3. 尝试复述内容

**推荐内容：**
1. SwiftUI Complete Tutorial
2. Networking in Swift
3. Core Data Tutorial
4. Swift Concurrency

---

### 高级阶段（5月以后）

**每日任务：**
1. 看1个WWDC Session（40-60分钟）
2. 学习专业术语
3. 尝试做英文笔记

**推荐内容：**
1. Stanford CS193p
2. WWDC Advanced Sessions
3. Small Advantages面试系列

---

## 观看技巧

### 如何利用视频学英语

1. **第一遍**：中英文字幕，了解内容
2. **第二遍**：只看英文字母，尝试理解
3. **第三遍**：不看字幕，专注于听
4. **第四遍**：跟读模仿
5. **做笔记**：记录常用表达和术语

### 实用表达记录模板

```

```
