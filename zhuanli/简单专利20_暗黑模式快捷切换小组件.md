# 专利20：一种暗黑模式快捷切换组件

## 技术领域
本发明涉及iOS主题切换技术领域，特别涉及一种通过通知中心快捷切换暗黑模式的小组件。

## 背景技术
用户需要在不同光照环境下切换深色/浅色模式，但现有方式需要进入设置 → 显示 → 黑暗模式，步骤繁琐。

## 发明内容

### 核心实现
```swift
// 1. 通知中心Widget
struct ThemeSwitchWidget: Widget {
    var body: some WidgetConfiguration {
        StaticConfiguration(kind: "ThemeSwitch", provider: ThemeProvider()) { entry in
            ThemeSwitchEntryView(entry: entry)
        }
    }
}

struct ThemeProvider: TimelineProvider {
    func getSnapshot(for configuration: ConfigurationAppIntent,
                    in context: Context,
                    completion: @escaping (ThemeEntry) -> Void) {

        let currentMode = UIScreen.main.traitCollection.userInterfaceStyle == .dark
        let entry = ThemeEntry(date: Date(), isDarkMode: currentMode)
        completion(entry)
    }
}

// 2. 快捷切换Intent
struct ToggleThemeIntent: AppIntent {
    static var title: LocalizedStringResource = "切换主题"

    func perform() async throws -> some IntentResult {
        let newMode = await MainActor.run {
            let current = UITraitCollection.current.userInterfaceStyle
            let new: UIUserInterfaceStyle = current == .dark ? .light : .dark

            // 更新全局样式
            if let windowScene = UIApplication.shared.connectedScenes.first as? UIWindowScene {
                windowScene.windows.forEach { $0.overrideUserInterfaceStyle = new }
            }

            return new
        }

        return .result()
    }
}

// 3. 创建切换快捷操作
extension ThemeSwitchWidget {
    struct ToggleButton: View {
        let isDark: Bool

        var body: some View {
            Button(intent: ToggleThemeIntent()) {
                Image(systemName: isDark ? "sun.max.fill" : "moon.fill")
                    .font(.title2)
                    .foregroundColor(isDark ? .yellow : .indigo)
            }
        }
    }
}
```

## 优点
1. 通知中心一键切换
2. 无需进入设置
3. 小组件直观显示当前模式
