# 专利10：一种通知栏快捷操作小组件

## 技术领域
本发明涉及iOS通知扩展技术领域，特别涉及一种在通知栏中集成快捷操作按钮的方法。

## 背景技术
车辆有异常状态时，推送通知只能显示文字，用户需要点击进入APP才能处理，操作链路长。

## 发明内容

### 核心实现
```swift
// Notification Service Extension
class NotificationService: UNNotificationServiceExtension {

    var contentHandler: ((UNNotificationContent) -> Void)?
    var bestAttemptContent: UNMutableNotificationContent?

    override func didReceive(_ request: UNNotificationRequest,
                            withContentHandler contentHandler: @escaping (UNNotificationContent) -> Void) {

        self.contentHandler = contentHandler
        bestAttemptContent = (request.content.mutableCopy() as? UNMutableNotificationContent)

        // 添加快捷操作按钮
        if let content = bestAttemptContent {
            content.categoryIdentifier = "VEHICLE_ALERT"

            // 定义快捷操作
            let unlockAction = UNNotificationAction(
                identifier: "UNLOCK_ACTION",
                title: "🔓 解锁",
                options: [.foreground]
            )

            let locateAction = UNNotificationAction(
                identifier: "LOCATE_ACTION",
                title: "📍 寻车",
                options: [.foreground]
            )

            let dismissAction = UNNotificationAction(
                identifier: "DISMISS_ACTION",
                title: "忽略",
                options: [.destructive]
            )

            let category = UNNotificationCategory(
                identifier: "VEHICLE_ALERT",
                actions: [unlockAction, locateAction, dismissAction],
                intentIdentifiers: []
            )

            UNUserNotificationCenter.current().setNotificationCategories([category])
        }

        contentHandler(bestAttemptContent!)
    }
}

// 处理快捷操作
class NotificationHandler {

    static func handle(actionIdentifier: String, notification: UNNotification) {
        let vehicleId = notification.request.content.userInfo["vehicle_id"] as? String ?? ""

        switch actionIdentifier {
        case "UNLOCK_ACTION":
            VehicleSDK.shared.unlock(vehicleId: vehicleId)

        case "LOCATE_ACTION":
            VehicleSDK.shared.flashLights(vehicleId: vehicleId)

        case "DISMISS_ACTION":
            // 静默忽略
            break

        default:
            break
        }
    }
}
```

## 优点
1. 通知栏直接处理，无需进入APP
2. 减少用户操作步骤
3. 提升异常处理效率
