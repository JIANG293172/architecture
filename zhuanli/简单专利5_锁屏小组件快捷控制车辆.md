# 专利5：一种锁屏小组件快捷控制车辆的方法

## 技术领域
本发明涉及iOS Widget开发技术领域，特别涉及一种在锁屏界面通过Widget快速控制车辆的方法。

## 背景技术
用户需要解锁车辆时，现有线控流程：掏出手机 → 解锁屏幕 → 打开APP → 点击解锁，步骤太多。

## 发明内容

### 核心实现
```swift
// iOS 16+ Lock Screen Widget
struct VehicleControlWidget: Widget {
    let kind: String = "VehicleControlWidget"

    var body: some WidgetConfiguration {
        StaticConfiguration(kind: kind, provider: Provider()) { entry in
            VehicleControlWidgetEntryView(entry: entry)
        }
        .configurationDisplayName("车辆控制")
        .description("快速解锁和启动车辆")
    }
}

struct Provider: TimelineProvider {
    func getSnapshot(for configuration: ConfigurationAppIntent,
                    in context: Context,
                    completion: @escaping (VehicleEntry) -> Void) {

        let entry = VehicleEntry(
            date: Date(),
            vehicleStatus: .locked,
            batteryLevel: 85
        )
        completion(entry)
    }

    func getTimeline(for configuration: ConfigurationAppIntent,
                    in context: Context,
                    completion: @escaping (Timeline<VehicleEntry>) -> Void) {
        // 提供AppIntent实现快捷操作
        let timeline = Timeline(entries: [VehicleEntry(...)], policy: .atEnd)
        completion(timeline)
    }
}

// 定义快捷操作Intent
struct UnlockVehicleIntent: AppIntent {
    static var title: LocalizedStringResource = "解锁车辆"

    func perform() async throws -> some IntentResult {
        // 调用车控SDK执行解锁
        await VehicleSDK.shared.unlock()
        return .result()
    }
}
```

## 优点
1. 锁屏直接操作，一步到位
2. 支持Face ID/Touch ID身份验证
3. 无需打开APP
