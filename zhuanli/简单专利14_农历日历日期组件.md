# 专利14：一种农历日期显示组件

## 技术领域
本发明涉及iOS日期组件技术领域，特别涉及一种同时显示公历和农历日期的组件。

## 背景技术
车控APP预约功能常需选择日期，但现有日历组件只显示公历，无法满足用户查看农历节日的需求。

## 发明内容

### 核心实现
```swift
class LunarCalendarView: UIView {

    private var currentMonth: Date = Date()
    private var lunarCalendar: [Int: String] = [:]  // day -> lunar string

    override init(frame: CGRect) {
        super.init(frame: frame)
        loadLunarData()
    }

    private func loadLunarData() {
        // 加载农历数据
        let calendar = Calendar(identifier: .chinese)
        let components = calendar.dateComponents([.year, .month, .day], from: currentMonth)

        // 获取当月农历信息
        let lunarInfo = LunarCalendarConverter.convert(
            year: components.year!,
            month: components.month!,
            day: components.day!
        )

        lunarCalendar = lunarInfo.days  // day: "初一", "初二", ...
    }

    func render() -> UIView {
        let container = UIStackView()
        container.axis = .vertical
        container.spacing = 8

        // 日期格子
        for day in 1...getDaysInMonth() {
            let dayCell = createDayCell(day: day)
            container.addArrangedSubview(dayCell)
        }

        return container
    }

    private func createDayCell(day: Int) -> UIView {
        let cell = UIView()

        // 公历日期
        let solarLabel = UILabel()
        solarLabel.text = "\(day)"

        // 农历日期
        let lunarLabel = UILabel()
        lunarLabel.text = lunarCalendar[day] ?? ""
        lunarLabel.font = .systemFont(ofSize: 10)
        lunarLabel.textColor = .secondaryLabel

        // 特殊日期高亮（如春节、中秋）
        if let lunar = lunarCalendar[day], isHoliday(lunar) {
            cell.backgroundColor = .systemRed.withAlphaComponent(0.1)
            lunarLabel.textColor = .systemRed
        }

        cell.addSubview(solarLabel)
        cell.addSubview(lunarLabel)

        return cell
    }

    private func isHoliday(_ lunar: String) -> Bool {
        let holidays = ["除夕", "春节", "元宵", "端午", "中秋"]
        return holidays.contains(lunar)
    }
}
```

## 优点
1. 同时显示公历农历
2. 农历节日自动高亮
3. 满足传统节日预约需求
