# 专利4：一种摇一摇快捷反馈提交组件

## 技术领域
本发明涉及iOS用户反馈技术领域，特别涉及一种通过摇一摇手势快速提交反馈的组件。

## 背景技术
现有反馈功能入口深，需要：设置 → 帮助 → 反馈 → 输入 → 提交，步骤繁琐，用户往往放弃反馈。

## 发明内容

### 核心实现
```swift
class ShakeFeedbackHandler {

    private var feedbackWindow: FeedbackPopupWindow?

    func startListening() {
        // 监听摇一摇
        UIWindow.startMotion()
    }

    override func motionEnded(_ motion: UIEvent.EventSubtype, with event: UIEvent?) {
        if motion == .motionShake {
            // 检测到摇一摇，弹出反馈窗口
            showFeedbackPopup()
        }
    }

    private func showFeedbackPopup() {
        // 创建半屏弹窗
        let popup = FeedbackPopupView()
        popup.show {
            // 可选：自动截取当前屏幕
            popup.attachScreenshot(screenshot: captureScreen())
        }
    }

    private func captureScreen() -> UIImage? {
        // 截取当前页面
        return UIGraphicsGetImageFromCurrentImageContext()
    }
}

// 反馈弹窗视图
class FeedbackPopupView: UIView {

    var onSubmit: ((String, UIImage?) -> Void)?

    // 快速输入 + 截图预览 + 一键提交
}
```

## 优点
1. 摇一摇即可触发，操作极简
2. 自动附带截图，减少用户操作
3. 弹窗式设计，不打断当前操作

## 权利要求
1. 一种摇一摇快捷反馈提交组件，其特征在于：检测到摇一摇手势时弹出反馈窗口。
2. 根据权利要求1所述的组件，其特征在于：自动截取当前屏幕并附带到反馈内容中。
