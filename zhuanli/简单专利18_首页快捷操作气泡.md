# 专利18：一种首页快捷操作气泡组件

## 技术领域
本发明涉及iOS首页组件技术领域，特别涉及一种可自定义位置的气泡式快捷操作组件。

## 背景技术
首页功能入口多时，图标密集，用户难以快速找到常用功能，且不同用户常用功能不同。

## 发明内容

### 核心实现
```swift
class BubbleShortcutView: UIView {

    private var isExpanded = false
    private var mainButton: UIButton!
    private var childButtons: [UIButton] = []

    override init(frame: CGRect) {
        super.init(frame: frame)
        setupMainButton()
    }

    private func setupMainButton() {
        mainButton = UIButton(type: .system)
        mainButton.setImage(UIImage(systemName: "plus"), for: .normal)
        mainButton.backgroundColor = .systemBlue
        mainButton.tintColor = .white
        mainButton.layer.cornerRadius = 28
        mainButton.addTarget(self, action: #selector(toggleExpand), for: .touchUpInside)
        addSubview(mainButton)
    }

    @objc private func toggleExpand() {
        isExpanded.toggle()

        UIView.animate(withDuration: 0.3, delay: 0,
                      usingSpringWithDamping: 0.7,
                      initialSpringVelocity: 0.5) {

            if self.isExpanded {
                // 展开子按钮
                self.showChildButtons()
                self.mainButton.transform = CGAffineTransform(rotationAngle: .pi / 4)
            } else {
                self.hideChildButtons()
                self.mainButton.transform = .identity
            }
        }
    }

    private func showChildButtons() {
        let angleStep: CGFloat = .pi / 4  // 45度间隔
        let radius: CGFloat = 80

        for (index, button) in childButtons.enumerated() {
            let angle = CGFloat(index) * angleStep - .pi / 2
            let x = cos(angle) * radius
            let y = sin(angle) * radius

            button.transform = CGAffineTransform(translationX: x, y: y)
            button.alpha = 1
        }
    }

    func setChildActions(_ actions: [(icon: String, action: () -> Void)]) {
        for (icon, handler) in actions {
            let button = UIButton(type: .system)
            button.setImage(UIImage(systemName: icon), for: .normal)
            button.backgroundColor = .systemGray5
            button.tintColor = .systemBlue
            button.layer.cornerRadius = 22
            button.alpha = 0

            button.addAction(UIAction { _ in
                handler()
                self.toggleExpand()
            }, for: .touchUpInside)

            childButtons.append(button)
            addSubview(button)
        }
    }
}
```

## 优点
1. 气泡展开式设计，节省空间
2. 用户自定义常用功能
3. 动画流畅自然

2. 根据权利要求1所述的组件，其特征在于：子按钮以扇形排列，采用弹性动画展开。
