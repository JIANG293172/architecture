# 专利8：一种滑动返回时显示操作菜单的方法

## 技术领域
本发明涉及iOS导航交互技术领域，特别涉及一种在滑动返回过程中显示上下文操作菜单的方法。

## 背景技术
用户在列表页编辑某项后想快速删除，通常需要进入详情页再返回，操作繁琐。

## 发明内容

### 核心实现
```swift
class SwipeActionHandler: NSObject {

    func setupSwipeAction(on cell: UITableViewCell) {
        let swipe = UISwipeGestureRecognizer(target: self, action: #selector(handleSwipe(_:)))
        swipe.direction = .right
        cell.addGestureRecognizer(swipe)
    }

    @objc private func handleSwipe(_ gesture: UISwipeGestureRecognizer) {
        guard let cell = gesture.view as? UITableViewCell,
              let tableView = cell.superview as? UITableView else { return }

        // 识别是哪一行
        guard let indexPath = tableView.indexPath(for: cell) else { return }

        // 显示操作菜单
        showActionSheet(from: cell, indexPath: indexPath)
    }

    private func showActionSheet(from cell: UITableViewCell, indexPath: IndexPath) {
        let alert = UIAlertController(title: nil, message: nil, preferredStyle: .actionSheet)

        alert.addAction(UIAlertAction(title: "删除", style: .destructive) { [weak self] _ in
            self?.deleteItem(at: indexPath)
        })

        alert.addAction(UIAlertAction(title: "收藏", style: .default) { _ in
            // 收藏操作
        })

        alert.addAction(UIAlertAction(title: "取消", style: .cancel))

        // 弹出菜单
        if let vc = UIApplication.shared.windows.first?.rootViewController {
            vc.present(alert, animated: true)
        }
    }
}
```

## 优点
1. 右滑直接呼出操作菜单
2. 无需进入详情页
3. 操作效率大幅提升
