# 专利16：一种短信验证码自动填充方法

## 技术领域
本发明涉及iOS验证码技术领域，特别涉及一种自动识别短信内容并填充验证码的方法。

## 背景技术
APP登录需要输入短信验证码，用户需要：收到短信 → 切回APP → 记住号码 → 输入，步骤繁琐。

## 发明内容

### 核心实现
```swift
// 利用iOS 12+ Automatic Verification
// 首先在项目中启用 "Automatic Verification"

class VerificationCodeManager: NSObject {

    static let shared = VerificationCodeManager()

    func startListening() {
        // 注册短信内容观察
        NotificationCenter.default.addObserver(
            self,
            selector: #selector(handleSMSReceived),
            name: NSNotification.Name("SMSReceived"),
            object: nil
        )
    }

    @objc private func handleSMSReceived(_ notification: Notification) {
        guard let smsContent = notification.userInfo?["content"] as? String else { return }

        // 提取验证码
        if let code = extractVerificationCode(from: smsContent) {
            // 自动填充到当前焦点输入框
            autoFillCode(code)
        }
    }

    private func extractVerificationCode(from sms: String) -> String? {
        // 匹配常见验证码格式：4-6位数字
        let pattern = "(?<!\\d)\\d{4,6}(?!\\d)"
        guard let regex = try? NSRegularExpression(pattern: pattern),
              let match = regex.firstMatch(in: sms, range: NSRange(sms.startIndex..., in: sms)),
              let range = Range(match.range, in: sms) else {
            return nil
        }

        return String(sms[range])
    }

    private func autoFillCode(_ code: String) {
        // 找到当前焦点输入框并填充
        DispatchQueue.main.async {
            if let firstResponder = self.findCurrentFirstResponder() as? UITextField {
                firstResponder.text = code

                // 触发值改变事件
                firstResponder.sendActions(for: .editingChanged)

                // 可选：自动提交
                self.submitIfReady()
            }
        }
    }
}

// iOS 16+ 短信链接自动验证（无需输入）
@available(iOS 16.0, *)
class AutomaticVerificationHandler {

    func handleAutomaticVerification(response: URL) async -> Bool {
        // 处理来自短信运营商的自动验证回调
        // 用户无需输入验证码
        return true
    }
}
```

## 优点
1. 验证码自动识别并填充
2. 无需用户手动切换应用
3. 支持iOS 16+自动验证（免输验证码）
