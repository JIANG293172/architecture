# 专利7：一种双指捏合快速缩放列表密度的方法

## 技术领域
本发明涉及iOS列表交互技术领域，特别涉及一种通过双指捏合手势快速调整列表密度的组件。

## 背景技术
用户在不同场景需要不同列表密度：开车时需要大按钮，停车时需要看到更多内容。但现有列表不支持动态调整密度。

## 发明内容

### 核心实现
```swift
class FlexibleListView: UITableView {

    private var rowHeightMultiplier: CGFloat = 1.0
    private let minMultiplier: CGFloat = 0.5
    private let maxMultiplier: CGFloat = 2.0

    override init(frame: CGRect, style: UITableView.Style) {
        super.init(frame: frame, style: style)
        setupPinchGesture()
    }

    private func setupPinchGesture() {
        let pinch = UIPinchGestureRecognizer(target: self, action: #selector(handlePinch(_:)))
        addGestureRecognizer(pinch)
    }

    @objc private func handlePinch(_ gesture: UIPinchGestureRecognizer) {
        switch gesture.state {
        case .changed:
            let newMultiplier = rowHeightMultiplier * gesture.scale

            // 限制范围
            if newMultiplier >= minMultiplier && newMultiplier <= maxMultiplier {
                rowHeightMultiplier = newMultiplier

                // 动态更新行高
                UIView.animate(withDuration: 0.2) {
                    self.beginUpdates()
                    self.endUpdates()
                }
            }

            gesture.scale = 1.0

        default:
            break
        }
    }

    override func tableView(_ tableView: UITableView, heightForRowAt indexPath: IndexPath) -> CGFloat {
        let baseHeight: CGFloat = 60
        return baseHeight * rowHeightMultiplier
    }
}
```

## 优点
1. 自然手势操作，用户学习成本低
2. 实时响应，无延迟感
3. 满足不同场景需求
