# 专利2：一种车辆状态卡片的3D翻转交互方法

## 技术领域
本发明涉及iOS交互设计领域，特别涉及一种用于展示车辆状态详情的3D翻转卡片组件。

## 背景技术
车控APP需要展示车辆状态信息（位置、电量、里程等），现有方案采用 Tab 切换或滚动列表，视觉单调，无法给用户直观的"查看详情"感受。

## 发明内容

### 核心实现
```swift
class VehicleCardView: UIView {

    private let frontView = UIView()   // 正面：概览
    private let backView = UIView()    // 背面：详情

    private var isShowingBack = false

    override init(frame: CGRect) {
        super.init(frame: frame)
        setupViews()
        setupGesture()
    }

    private func setupViews() {
        // 正面：车辆图标 + 电量 + 位置
        frontView.backgroundColor = .systemBlue
        addSubview(frontView)

        // 背面：详细信息
        backView.backgroundColor = .systemIndigo
        backView.isHidden = true
        addSubview(backView)
    }

    private func setupGesture() {
        let tap = UITapGestureRecognizer(target: self, action: #selector(flipCard))
        addGestureRecognizer(tap)
    }

    @objc private func flipCard() {
        // 3D翻转动画
        let options: UIView.AnimationOptions = [
            .transitionFlipFromRight,
            .showHideTransitionViews
        ]

        UIView.transition(
            with: self,
            duration: 0.5,
            options: options
        ) {
            self.frontView.isHidden = self.isShowingBack
            self.backView.isHidden = !self.isShowingBack
        } completion: { _ in
            self.isShowingBack.toggle()
        }
    }
}
```

## 优点
1. 3D翻转效果直观展示详情
2. 单手操作便捷
3. 视觉效果突出，品牌感强
