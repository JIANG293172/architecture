# 专利1：一种基于长按手势的图片预览方法

## 技术领域
本发明涉及iOS图片交互技术领域，特别涉及一种通过长按手势快速预览图片的方法。

## 背景技术
现有图片预览需要：点击图片 → 进入新页面 → 查看 → 返回，步骤繁琐。用户只是想快速看一眼图片内容，却需要完整跳转页面，体验不佳。

## 发明内容
本发明提供一种长按手势图片预览方法：

### 核心实现
```swift
class ImagePreviewGestureHandler {

    private var previewController: UIViewController?

    // 长按手势触发快速预览
    func setupLongPressPreview(on imageView: UIImageView) {
        let longPress = UILongPressGestureRecognizer(
            target: self,
            action: #selector(handleLongPress(_:))
        )
        longPress.minimumPressDuration = 0.3  // 300ms
        imageView.addGestureRecognizer(longPress)
        imageView.isUserInteractionEnabled = true
    }

    @objc private func handleLongPress(_ gesture: UILongPressGestureRecognizer) {
        switch gesture.state {
        case .began:
            // 显示半透明预览浮层
            showPreview(at: gesture.location(in: gesture.view?.superview))

        case .ended, .cancelled:
            // 手指离开隐藏预览
            hidePreview()

        default:
            break
        }
    }

    private func showPreview(at point: CGPoint) {
        // 创建半透明预览视图
        let previewView = createPreviewView()
        previewView.frame = CGRect(
            x: point.x - 100,
            y: point.y - 75,
            width: 200,
            height: 150
        )

        // 添加模糊背景
        let blurEffect = UIBlurEffect(style: .dark)
        let blurView = UIVisualEffectView(effect: blurEffect)
        blurView.frame = UIScreen.main.bounds
        previewView.addSubview(blurView)

        UIApplication.shared.windows.first?.addSubview(previewView)
        self.previewController = previewView
    }

    private func hidePreview() {
        UIView.animate(withDuration: 0.2) {
            self.previewController?.alpha = 0
        } completion: { _ in
            self.previewController?.removeFromSuperview()
        }
    }
}
```

## 优点
1. 操作步骤从3步简化为1步
2. 无需跳转页面，原地预览
3. 轻量化实现，不占用页面栈
