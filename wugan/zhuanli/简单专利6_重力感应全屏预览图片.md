# 专利6：一种基于重力感应的全屏图片预览方法

## 技术领域
本发明涉及iOS图片预览技术领域，特别涉及一种通过倾斜设备实现图片全屏预览的方法。

## 背景技术
图片预览时需要点击按钮才能全屏，而用户只是想无遮挡地快速查看。

## 发明内容

### 核心实现
```swift
class TiltFullscreenHandler: UIViewController {

    private var originalFrame: CGRect = .zero
    private var isFullscreen = false
    private var motionManager: CMMotionManager?

    func setupTiltDetection() {
        motionManager = CMMotionManager()
        motionManager?.deviceMotionUpdateInterval = 0.1

        motionManager?.startDeviceMotionUpdates(to: .main) { [weak self] motion, _ in
            guard let motion = motion else { return }

            let pitch = motion.attitude.pitch

            // 倾斜超过45度触发全屏
            if pitch > .pi / 4 && !self!.isFullscreen {
                self?.enterFullscreen()
            } else if pitch < .pi / 6 && self!.isFullscreen {
                self?.exitFullscreen()
            }
        }
    }

    private func enterFullscreen() {
        originalFrame = imageView.frame

        UIView.animate(withDuration: 0.3) {
            self.imageView.frame = UIScreen.main.bounds
            self.imageView.contentMode = .scaleAspectFit
            self.view.backgroundColor = .black
        }
        isFullscreen = true
    }

    private func exitFullscreen() {
        UIView.animate(withDuration: 0.3) {
            self.imageView.frame = self.originalFrame
            self.view.backgroundColor = .white
        }
        isFullscreen = false
    }
}
```

## 优点
1. 无需点击，自然交互
2. 倾斜角度触发，误触率低
3. 退出同样自然

2. 根据权利要求1所述的方法，其特征在于：当设备恢复接近水平时自动退出全屏。
